# Changelog

All notable changes to **SocialPreview Dashboard** will be documented in this file.

## [1.0.0] - 2025-12-29

### Added
- **Initial Release**
- **Rust Backend**: Axum-based high-performance API.
- **React Frontend**: Modern SPA with Vite and Tailwind CSS.
- **Ticketing System**: Support for creating, managing, and commenting on tickets.
- **Task Management**: Team-oriented task assignments and tracking.
- **RBAC**: Full role-based access control (Management, Team, User).
- **2FA**: TOTP support for enhanced security.
- **Stats & News**: Integrated dashboard statistics and internal blog system.
- **Documentation**: Extensive technical documentation in the `docs/` directory.
